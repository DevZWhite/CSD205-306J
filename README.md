# Python CSD205-306J

## Overview

This repository contains source code and exercises focused on **learning Python** as part of a programming course at **Bellevue University**. The purpose of this repository is to demonstrate foundational to intermediate Python concepts through hands-on practice, reinforcing core programming principles such as problem solving, clean code structure, and documentation.

The project emphasizes writing readable, maintainable Python code while gradually introducing more advanced language features.

## Contributors

* **Instructor:** Darrell Payne

## Learning Objectives

Through the projects and exercises in this repository, the following concepts are explored:

* Python syntax and best practices
* Variables and data types
* Conditional logic (selection)
* Loops and repetition
* Functions and modular design
* Input and output operations
* File I/O
* Error and exception handling
* String manipulation and formatting
* Basic object-oriented programming in Python
* Writing clear comments and documentation

## Project Structure

The repository may include:

* Individual Python scripts (`.py`) demonstrating specific concepts
* Small programs built from pseudocode and flowcharts
* Incremental assignments that build on previous knowledge
* Well-commented examples for learning and reference
